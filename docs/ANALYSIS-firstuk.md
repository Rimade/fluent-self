# Анализ референса firstuk.school

> Дата: июнь 2026 · Для проекта **Fluent Self** (взрослые, пастель, проще kids-сайта)

---

## 1. Общее впечатление

[First UK School](https://firstuk.school/) — **премиальный минималистичный** сайт языковой школы. Студия [Redo Bureau](http://redobureau.com). Не «яркий лендинг», а **editorial**: много воздуха, serif-заголовки, чёрно-белая база, акцент — приглушённый зелёный `#4b7d72`.

**Для Fluent Self:** берём **структуру блоков и смысл контента**, но **не копируем** тёмную luxury-эстетику — нужны **пастельные спокойные** тона и более простая вёрстка без тяжёлых scroll-анимаций.

## 4. Главная страница — секции сверху вниз

### 4.1 Hero (full viewport)

- Высота `100vh`, центр экрана
- **TypeIt** — поочерёдная анимация крупных фраз (uppercase)
- Внизу — перевод / подпись курсивом (italic serif)
- Минимум декора, белый фон

**Fluent Self:** статичный заголовок «Школа иностранных языков для взрослых» + подзаголовок + CTA «Пробный урок бесплатно» — без TypeIt.

### 4.2 «Наши известные ученики»

- **Desktop:** split-screen — слева фото выбранного человека, справа прокручиваемый список имён (scroll-sync)
- **Mobile:** Swiper с портретами
- Знаменитости: телеведущие, актёры, музыканты + подпись профессии

**Fluent Self:** **убрать** блок знаменитостей (у нового бренда нет) → заменить на **отзывы**, **преподавателей** или **«почему мы»** в 3–4 пункта.

### 4.3 «Наши языковые курсы»

Три **квадратные карточки** (на desktop первая шире):

| Карточка | Цвет-точка | Текст |
|----------|------------|-------|
| Курсы для взрослых | `#060B11` (почти чёрный) | Английский, немецкий + ещё 5 языков |
| Курсы для детей | `#C1CFCF` (sage) | Английский 3–16 лет |
| Курсы для бизнеса | `#571A2A` (burgundy) | Для сотрудников |

Карточка = border-сетка, hover, ссылка «Подробнее →».

**Fluent Self:** 2–3 карточки в **пастельных** цветах (мятный, пудровый, лавандовый). Детскую — опционально.

### 4.4 Parallax-фото

- Full-width фон интерьера школы (`background-attachment: fixed` на desktop)

**Fluent Self:** обычная секция с фото + подпись, без parallax (проще и быстрее).

### 4.5 Преимущества (9 пунктов)

1. Уютные кабинеты
2. Зона ожидания
3. Носители языка
4. Поездки за границу
5. Бесплатная парковка
6. Сертификаты
7. Небольшие группы (до 6)
8. Пробный урок
9. Удобный график

- **Mobile:** accordion (expander)
- **Desktop:** тот же scroll-sync, что у «учеников» — список справа, контент слева + номер в круге

**Fluent Self:** сетка **3×3** или **2×4** карточек с иконками — без scroll-sync.

### 4.6 Отзывы

- Карусель имён (Swiper)
- Текст отзыва по центру, крупный serif/italic
- ~5 реальных отзывов про атмосферу и преподавателей

**Fluent Self:** оставить, можно упростить до 3 карточек без автопрокрутки.

### 4.7 Footer / контакты

- Яндекс.Карта (iframe)
- Адрес, метро, график 10:00–20:00
- Телефон, соцсети (Facebook, Instagram, VK)
- Юрлицо (ООО, ИНН)
- WhatsApp-плавающая кнопка

## 7. Дизайн-система firstuk (что не копируем буквально)

| Элемент | firstuk | Fluent Self (целевое) |
|---------|---------|------------------------|
| Фон | Белый `#fff` | Тёплый off-white `#faf8f5` |
| Текст | Чёрный | Мягкий charcoal `#3d4a47` |
| Акцент ссылок | `#4b7d72` | Пастельный teal `#7aab9e` |
| Карточки курсов | Тёмные dot-цвета | Мятный, персик, лаванда |
| Шрифты | Times New Roman + Graphik | Fraunces/Lora + Libre Franklin или Source Sans (не Inter/Roboto) |
| Hero | TypeIt fullscreen | Компактный hero + иллюстрация |
| Анимации | Scroll-sync, parallax, Swiper | Лёгкий fade, один слайдер |

**Пастельная палитра (черновик):**

```css
--bg: #faf8f5;
--bg-soft: #f3efea;
--primary: #7aab9e;      /* мятный */
--primary-soft: #dceee8;
--accent: #e8cfc4;       /* персик */
--accent-2: #d4cfe8;     /* лаванда */
--ink: #3d4a47;
--muted: #7a8582;
--line: #e8e4df;
```

## 8. Технический стек firstuk

- Кастомный CSS (~750 KB minified), grid flex
- JS: TypeIt, Swiper 4, scroll behaviors, Google Maps API
- CMS-подобная структура (media в `/media/pages/`)
- `noindex, nofollow` в meta (технически странно для prod)

**Fluent Self:** статика как kids-проект — partials через fetch, без CMS, минимальный JS.

## 10. Рекомендуемый scope MVP для Fluent Self

**Обязательно:**
1. Главная (hero, 2–3 курса, 6–8 преимуществ, отзывы, CTA)
2. Курсы для взрослых (список языков/программ)
3. О школе
4. Контакты + карта
5. Запись на пробный урок + форма/Telegram
6. Privacy (минимально)

**По желанию:**
- Корпоративное обучение
- Мероприятия / фото
- Отдельные страницы языков

**Не делать в v1:**
- Блок знаменитых учеников
- Scroll-sync анимации
- Parallax
- Детские курсы (если заказчик не попросит)
