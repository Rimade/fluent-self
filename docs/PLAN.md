## Упрощаем относительно firstuk

| firstuk | Fluent Self |
|---------|-------------|
| TypeIt hero на весь экран | Статичный hero + подзаголовок |
| Scroll-sync «знаменитые ученики» | Убрать или заменить на «наши студенты» / отзывы |
| Scroll-sync преимущества (9 пунктов) | Сетка карточек или простой аккордеон |
| Swiper + кастомный JS bundle | Минимум: 1 слайдер отзывов или статичный блок |
| Parallax фото | Обычное фото-секция |
| CMS / Kirby | Статический контент в `content.js` |

## Оценка объёма

**Проще kids-сайта:** ~8–12 страниц против 20+, меньше каталогов и предметных landing’ов.

**Срок ориентир:** 1–2 итерации после согласования макета главной.
